==========
ESP-Prog-2
==========

:link_to_translation:`zh_CN:[中文]`

ESP-Prog-2 is a versatile debugging tool developed by Espressif. It supports automatic firmware downloading, serial communication, and JTAG online debugging, and is compatible with various Espressif chips. With a simple design and flexible power options, ESP-Prog-2 provides reliable power and interface support, making it ideal for developing and debugging Espressif-based applications.
