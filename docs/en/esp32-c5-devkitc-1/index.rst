ESP32-C5-DevKitC-1
==================

:link_to_translation:`zh_CN:[中文]`

ESP32-C5-DevKitC-1 is an entry-level development board based on the general-purpose module ESP32-C5-WROOM-1. This board integrates complete Wi-Fi, Bluetooth LE, Zigbee, and Thread functions.
