ESP-WROVER-KIT
==============

:link_to_translation:`zh_CN:[中文]`

ESP-WROVER-KIT is an ESP32-based development board produced by `Espressif <https://espressif.com>`_. ESP-WROVER-KIT features the ESP32-WROVER-E module, LCD screen, and microSD card slot.
