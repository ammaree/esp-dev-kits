ESP32-PICO-KIT-1
================

:link_to_translation:`zh_CN:[中文]`

The :ref:`ESP32-PICO-KIT-1 <esp32-pico-kit-1-overview>` is an ESP32-based development board produced by `Espressif <https://www.espressif.com/>`_. ESP32-PICO-KIT-1 provides users with hardware for development of applications based on ESP32, making it easier for users to explore ESP32 functionalities.
