ESP-Module-Prog-1(R)
====================

:link_to_translation:`en:[English]`

ESP-Module-Prog-1 和 ESP-Module-Prog-1R 是乐鑫专为模组设计的两款烧录底板，无需将模组与电源和信号线焊接便可进行烧录。两款底板仅在弹簧布局上有所不同，用于适配不同的模组。

ESP-Module-Prog-1 适配 ESP32、ESP32-S2、ESP32-S3、ESP32-C6 芯片系列的部分模组，而 ESP-Module-Prog-1R 则专门适配 ESP32-WROVER 系列模组。具体底板对应的适配模组，请参见 :ref:`适配模组列表 <fitting-modules-of-prog-1>`。
