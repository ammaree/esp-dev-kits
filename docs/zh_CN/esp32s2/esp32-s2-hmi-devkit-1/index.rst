ESP32-S2-HMI-DevKit-1
========================

:link_to_translation:`en:[English]`

ESP32-S2-HMI-DevKit-1 面向 GUI 应用场景设计，可实现智能家居交互面板，带显示屏的音箱、闹钟等人机交互界面的智能控制。

开发板的应用示例存放在 :project:`Examples <examples/esp32-s2-hmi-devkit-1/examples>` 文件夹中。
